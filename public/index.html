Airtable JSON Ops Console

Configure everything from here: jobs, schedules, retention, run history, snapshots, and rollbacks. No Airtable API keys and no CLI setup required for day-to-day use.

Quick Setup

  1. Set default schedule and retention in Workspace Settings.
  2. Create one or more jobs with Airtable URL and output JSON path.
  3. Run Preview to verify extraction, then Run now to generate output.
  4. Leave schedule enabled to refresh data automatically every day.

Workspace Settings

Example: 0 17 * * * (17:00 daily).
